ADB HW_1

1. Отобразить подключённый девайс в консоли: adb devices
2. Вывести адрес приложения todolist в системе Android: adb shell 'pm list packages -f' |findstr(grep) todolist
3. Установить .apk файл приложениия todolist на телефон с компьютера через  ADB: adb install "E:\AndroidApp\app\build\outputs\apk\debug\app-debug.apk"
4. Сделать скриншот запущенного приложения todolist и сразу скопировать на компьютер в одной команде: adb shell screencap /storage/emulated/0/DCIM/screen.png && adb pull /storage/emulated/0/DCIM/screen.png C:\Users\Laptop1\Desktop\platform-tools\screen.png
5. Вывести в консоль логи приложения todolist: adb logcat|grep -rnw(select-string) "com.andoid.todolist"
6. Скопировать логи приложения todolist на компьютер: adb logcat|grep -rnw(select-string) "com.android.todolist" > todolist.log
7. Удалить приложение todolist с телефона через ADB: adb uninstall com.android.todolist
